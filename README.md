## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {lsci, ".*",
      {git, "git@github.com:YOURNAME/lsci.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
